# Y6 — Completable in one cycle (retrospective)

**Lane:** Upstream · **Owner:** Yanna · **Evidence type:** Deterministic

## What it checks

Actual cycle time from `In Implementation → Done Implementing` is ≤ 7 calendar days.

Note: The forward-looking prediction at commit time is recorded by C3 (gate approver). Y6 checks whether the prediction held.

## Why it matters

Nolte's 95% forecast accuracy depends on consistent sizing. Consistent sizing depends on the one-cycle rule holding. When Stories routinely exceed one cycle, the forecast is running on guesses.

## How the grader evaluates it

Deterministic. Reads timestamps of `In Implementation` transition and `Done Implementing` transition from changelog. Elapsed calendar days > 7 = fail.
