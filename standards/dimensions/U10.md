# U10 — Impact measurement infrastructure at Done

**Lane:** Upstream · **Owner:** Yanna · **Evidence type:** Deterministic

## What it checks

At Done, the `Impact Measurement Link` custom field is populated with a resolvable URL pointing to where impact will be observed (dashboard, query, document).

## Why it matters

Y2 defines *how we will know* the work worked. U10 makes sure the place where we check that knowledge actually exists. Without it, the "not yes people" posture has nowhere to look post-delivery — we ship and forget.

## How the grader evaluates it

Deterministic. Reads `Impact Measurement Link` custom field at Done. An optional non-blocking HTTP HEAD with 5s timeout verifies reachability.

## Evidence codes

| Code | Verdict | Meaning |
|---|---|---|
| `IMPACT_LINK_PRESENT_AND_RESOLVABLE` | PASS | URL present and reachable. |
| `IMPACT_LINK_PRESENT_NOT_VERIFIED` | PASS | URL present; reachability skipped. |
| `IMPACT_LINK_UNREACHABLE` | PASS (flagged) | URL present but transient fetch failure. |
| `IMPACT_LINK_MISSING` | FAIL | Field empty. |
| `IMPACT_LINK_BROKEN` | FAIL | URL resolves 4xx/5xx. |
